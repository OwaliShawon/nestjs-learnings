# Step 01: Introduction

## Goal
Understand what NestJS is and how this learning path is structured.

## What is NestJS?
NestJS is a progressive Node.js framework for building efficient, scalable server-side applications. It uses TypeScript by default and is built around modules, controllers, and providers.

## Why NestJS?
- Opinionated architecture
- Built-in dependency injection
- Great TypeScript support
- Works well for monoliths and microservices

## Learning path
You’ll build from basics (controllers, providers) to advanced topics (testing, microservices, deployment).

## Exercises
- Read the official NestJS docs overview.
- Identify one reason you’d choose NestJS for a backend project.
